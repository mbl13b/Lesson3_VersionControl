Highlight #1 Version Control System

The Version Control System automatically takes note of: the revision history, when they were revised, what about them is new, and any changes are merged to the same file. Benefits of this method are that files can't be overwritten, there is a common repository that holds all the latest files, and people can work on the same files simultaneously without conflict.

Highlight #2 Repositories

There are two types of repositories: the remote repository and the local repository. The remote repository resides on a remote server that is shared among multiple team members. A local repository resides on a local machine of an individual user and can be created by either making a brand new repository or by cloning an existing remote repository onto your local machine.

Highlight #3 Git

To start, a Git is a free and open source distributed VCS designed to handle everything with speed and efficiency. Git is the best type because every user has a complete copy of the repository data stored locally on their machine, everything is local to work off of, there is no single point of failure, processes run exceedingly fast, and Git will prevent an unintentional overwrite of a file on the shared repository.
